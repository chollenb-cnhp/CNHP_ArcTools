**EIA Summarizer for SNHS S123 Ecology Records**

**How to use:**

Option 1:
    -Download EIA_summarizer.pyt and put in new folder in working directory for GIS
    -Open your existing SNHS ArcPro Project
    -Open Catalog Pane
    -Add Folder Connection to EIA_summarizer.pyt folder
    -Double click EIA_summarizer.pyt
    -Double click Process EIA
    -Select CNHP_Ecology_Field_Observations/CNHP_Ecology_Field_Observations as input feature class
    -Enter optional excel file path if export is desired
    -Review generated Standalone Table
    -Calculate fields or perform joins with CNHP_Ecology_Field_Observations to update Landscape and Condition Comments after review

Option 2 (Advanced)
    -Pull git repository to working GIS folder
    -Allows for updates to EIA_summarizer when I create code modifications
